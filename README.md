# First Website - Clases de Física y Matemática

Sitio web desarrollado como proyecto personal para ofrecer clases particulares de Física y Matemática.

El objetivo de este proyecto es practicar y consolidar conocimientos en HTML, CSS y JavaScript, construyendo una landing page real, responsive e interactiva.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- ScrollReveal
- Font Awesome

## Funcionalidades actuales

- Hero principal con llamada a la acción
- Menú de navegación responsive
- Menú hamburguesa con overlay
- Sección de estadísticas con diseño adaptable
- Formulario de contacto desplegable
- Scroll suave hacia secciones
- Validación básica de formulario
- Diseño responsive para distintas pantallas
